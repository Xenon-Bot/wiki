# Export Server

Export the server settings, channels, roles, emojis, and stickers as a JSON file.

## Syntax

`/export guild`

<br />

# Export Channels

Export a list of channels from this server.

## Syntax

`/export channels [format]`

## Arguments

# Tabs {.tabset}
## format
The file format for the export. Either `json` or `csv`.
